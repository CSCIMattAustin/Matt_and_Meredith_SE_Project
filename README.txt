This is a tuner and metronome. The design currently offers options that perform minimal operations. A note to for tuning can be selected using the comboBox drop down menu labeled "Please Select a Note." You can play the note by selecting "Start" and stop the note by selecting "Stop." You can access the metronome by adjusting the slider to the preferred tempo, then selecting "Start." The tempo can be adjusted by sliding the bar at any time. To stop the metronome, select "Stop." A green circle will flash with the tempo! There is also currently a drop down menu labeled "Please Select a Note Value," but it is not fully functional currently.
